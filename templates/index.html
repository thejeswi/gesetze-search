This is a search used to find relavent German laws (Gesetze) to an input query.

Features

  • Search in German or English.
  • Results take you to specific paragraphs of the law.
  • Full logic operation support (e.g. "stock NOT market", etc).
  • Supports stemming of search terms (e.g. a search for "football" would match documents which mention "footballs" or "footballer").
  • Wildcard search is supported (e.g. "destr*" will match destroy, destruct, so on).
